# React app with client-server

## Installation

#### `git clone`
### `git clone https://github.com/Drew11/toweriq-test-app.git`
### `npm install`

To start dev servers use commands:
### `npm start` 
_starting dev server on `localhost:5000`_
### `npm run client` 
_starting client server on `localhost:3000`_
